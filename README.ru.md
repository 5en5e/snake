# Snake Game

## 📌 Что это
Классическая игра "змейка".
Управляется стрелками клавиатуры.

## ⚙️ Что умеет
- регулировка скорости
- предупреждение о потере фокуса
- пауза/запуск
- кнопка автоматической победы (cheat mode)

---

## ⚠️ Известные проблемы
- размер эмодзи еды зависит от высоты вьюпорта

---

## 🚧 Планируемые улучшения
- исправление проблем из пункта выше
- использование семантических HTML-тегов
- замена кастомных атрибутов на data-*
- добавление постепенного ускорения
- отображение длины змейки при окончании игры
- возможность интерактивно задать размер сетки
- разделение DOM и логики
